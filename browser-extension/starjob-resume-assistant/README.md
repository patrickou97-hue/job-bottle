# 拾星网申助手

拾星网申助手是拾星 StarJob 的 Manifest V3 浏览器扩展。它把用户主动同步到浏览器本地的拾星简历内容填写到当前网申页面。

## 安全边界

- 只在用户点击“填写当前页面”后临时访问当前标签页。
- 不读取浏览历史、Cookie、密码或账号 CSV。
- 简历保存在 `chrome.storage.local`，卸载扩展时由浏览器清除。
- 不自动点击提交，不填写身份证、性别、薪资、政治面貌、验证码等敏感字段。
- 文件上传、验证码、测评题和低置信度字段由用户手动完成。
- 智能匹配只上传去内容化的字段元数据，不上传输入框现有值或简历正文；服务端结果必须通过标准字段键白名单校验。

## 开发与打包

扩展目录本身即为可加载目录。仓库根目录执行：

```bash
npm run build:extension
```

脚本会校验必需文件并生成 `public/downloads/starjob-resume-assistant-v0.1.6.zip`，该 ZIP 可直接上传百度网盘。
